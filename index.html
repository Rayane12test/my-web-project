About The Game

Escape the Nightmare

Abyssborn is a first-person psychological horror game where you must escape a surreal illusion by collecting keys before time runs out. With a relentless stalker chasing you, every decision matters.

Set in a decaying hotel that shifts and changes around you, the game creates an atmosphere of constant dread and uncertainty. The environment itself is your enemy.

Developed using Godot Engine, Abyssborn features advanced AI behavior, dynamic lighting, and a haunting original soundtrack.

Game Features

Psychological Horror

Experience true psychological terror with mind-bending environments and unpredictable scares

Time Pressure

The clock is ticking - solve puzzles and find keys before time runs out

Advanced AI

The stalker learns from your behavior, adapting its hunting patterns

Dynamic Soundtrack

Original music that reacts to your actions and intensifies the horror

Smart Lighting

Realistic shadows and light effects that create an atmosphere of dread

Procedural Elements

Certain game elements change with each playthrough for replayability

Download Abyssborn

Before downloading Abyssborn, make sure your system meets the following requirements:

  • OS: Windows 10 (64-bit) or later
  • Processor: Intel Core i5 or AMD equivalent
  • Memory: 8 GB RAM
  • Graphics: NVIDIA GTX 960 or AMD Radeon R9 280
  • Storage: 20 GB available space
  • DirectX: Version 11

Steps to Download and Install

  1. Click the Download Now button below to download the game setup file.
  2. Once the download is complete, locate the file abyssborn-setup.exe in your downloads folder.
  3. Double-click the setup file to start the installation process.
  4. Follow the on-screen instructions to complete the installation.
  5. After installation, launch the game from your desktop or start menu.
